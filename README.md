# Airplane-crash-analysis-dashboard
An interactive dashboard analyzing global airplane crash trends, fatalities and crash locations

## Project Overview 
This project presents an interactive power BI  dashboard that analyzes historical airplane crash data. The dashboard provides insights into crash frequency over time, countries most affected, aircraft operators involved and overall fatality statistics.

## Tools Used
Microsoft Excel
Power BI 
Power Query 
DAX

## Key Insights 
Total airplane crashes recorded 
Number of passengers involved in crashes that lost their lives 
Crash occurences in a span of different years
